# prueba-coches
Spring Boot application

# How to Play

## Prerequires

1. Java 8+
2. Maven


## Clone

```
git clone https://github.com/carolinedesousasantos/prueba-coches.git
```

## Go into folder

```
cd prueba-coches
```

## Execute

```
mvn spring-boot:run
 ```

## To see api documentation

   [http://localhost:8080/swagger-ui/index.html](http://localhost:8080/swagger-ui/index.html)
